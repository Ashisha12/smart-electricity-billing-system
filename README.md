# Smart Electricity Billing System

This project is a Python-based application that calculates monthly electricity bills
based on units consumed using slab-wise tariff rates.

## Features
- Calculates electricity bill based on consumption slabs
- Uses real-world electricity pricing logic
- Handles invalid user input gracefully
- Displays a formatted electricity statement

## Technologies Used
- Python
- Conditional statements
- Functions
- Exception handling

## How to Run
1. Clone the repository
2. Run the file using:
   python bill_calculator.py
3. Enter the total units consumed when prompted

## Use Case
This project demonstrates how Python can be used to automate
basic electrical billing calculations in a real-world scenario.
